# Comp 401 Tutorial Website

# Description
This website is made for Comp 227 as a final project.
It is a tutorial website that helps students review and learn new concepts in computer science. 
I used Bootstrap templates from https://startbootstrap.com as the UI design. 

# [Start Bootstrap - Resume](https://startbootstrap.com/template-overviews/resume/)

[Resume](https://startbootstrap.com/template-overviews/resume/) is a resume and CV theme for [Bootstrap](http://getbootstrap.com/) created by [Start Bootstrap](http://startbootstrap.com/). This theme features a fixed sidebar with content sections to build a simple, yet elegant resume.
